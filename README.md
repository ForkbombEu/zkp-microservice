## 🏗️ Run your own

To run your own microservice run the following commands:
```
# clone the repository
git clone https://github.com/ForkbombEu/zkp-microservice.git
cd zkp-microservice

# install ncr and launch it
make up
```

Then visit http://localhost:3333/docs
